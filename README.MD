# webtrees module - Bulk delete objects

This module can delete many objects by their XREF.

Use with caution, at your own risk.

## Installation
Copy the files to directory `/modules_v4/wt-module-bulk-delete` on your webtrees server.

## Usage
 * First make a __backup__ of your data.
 * Navigate to the webtrees __Control Panel__
 * Select your tree
 * Select option __Data fixes__
 * Select __Bulk delete objects__ from drop-down and click __Next__
 * Paste the XREFs to delete in the text box
 * Click __Search__ when you are cautious or __Update all__ if you only live once (and have a backup)

Enjoy!
